# 🚨 APP Botón de Pánico - Frontend Admi

---

### 📦 Proyecto creado con:

- **Angular CLI:** 19.1.4  
- **Angular Material** 19.2.14  
- **Node:** 22.13.1  
- **Package Manager:** npm 10.9.2  
- **Ionic:** 7.2.0

## 🛠️ Instalación
```bash
git clone https://github.com/viklop-amiquero/btn_panic_front_admi.git
npm install
ng serve

```
